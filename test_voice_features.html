üìã Voice Commands Reference

English Commands:

  • start, begin, go, listen - Start voice recognition
  • stop, end, finish, quiet - Stop voice recognition
  • voice on, enable voice - Enable voice output
  • voice off, disable voice - Disable voice output
  • help, support, assist, commands - Show help information
  • repeat, again, once more, say again - Repeat last message
  • home, main, back, menu - Navigate to home
  • settings, options, preferences, configure - Open voice settings

Urdu Input Support:

üé§ Multi-Language Input: You can speak in Urdu and the system will understand your commands, even when English voice output is disabled. The system recognizes both English and Urdu speech input automatically.

üö® Common Issues & Solutions

Issue: "Speech Recognition not supported in this browser"
Solution: Use Chrome, Microsoft Edge, or Safari for full voice support. Firefox has limited support.
Issue: "Microphone permission denied"
Solution: Enable microphone access in browser settings (usually in site settings or privacy settings).
Issue: "No voices available"
Solution: Voices may take time to load. If this persists, check your browser's text-to-speech settings.
Issue: Voice commands not working
Solution: Speak clearly and ensure you're using the correct language commands. Check the reference above.
